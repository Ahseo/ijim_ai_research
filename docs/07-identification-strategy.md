# 식별 전략: 방법론 개선

## 현재 설계 평가

### PSM+DID/DDD가 제공하는 것
- **PSM**: 관측 가능한 선택 편의 처리 (수준 차이)
- **DID**: 시불변 비관측 차이 제거
- **DDD**: 3차원(숙련 그룹)과 상관된 교란 흡수

### 현재 강건성 검정
- 위약 검정 (도입 전 기간) — 유의한 효과 없음
- 대안 PSM 명세 (대체 포함, 다른 caliper, 대안 거리 측도)
- 주 단위 분석: 안정적 효과 확인

### 실제 약점 (리뷰어들이 맞음)

**1. 시변 비관측 선택 편의**
DID는 고정된 비관측 변수만 제거. 처리 시점에 차별적으로 변화하는 비관측 변수는 제거 불가.
위협 시나리오: "AI를 채택한 라이더는 AI와 무관하게 이미 상승 궤적에 있던 라이더들"
도입 전 수준에 대한 PSM은 궤적을 포착하지 못함.

**2. 이진 "ever adopter" 처리가 너무 조악함**
주문의 95%를 AI로 받는 라이더와 한 번만 써본 라이더를 동일시.
측정 오차 유발, 메커니즘 논증 약화.

**3. 짧은 기간 (전후 각 1개월)**
- 사전 추세 탐지 제한 (4주만)
- 신규성 효과와 영구 변화 구별 불가
- 시차별 채택 변동 제한

**4. 유효한 도구변수 없음**
AI 모드 접근에 외생적 변동 없음 (보편적 가용). IV/2SLS 불가.

## 신규 분석 (우선순위 순)

### 우선순위 1: Event-Study 플롯

**수식:**
```
Y_{it} = α_i + γ_t + Σ_{k=-4, k≠-1}^{+4} β_k(Treat_i × 1[t=k]) + X_{it}'δ + ε_{it}
```

**구현:**
- 주 단위 구간 (사전 4주, 사후 4주)
- k = -1 기준기간
- 라이더 수준 군집 표준오차

**통과 기준:**
- 모든 사전 처리 β_k가 개별적으로 5%에서 비유의
- 결합 F-검정 p > 0.10
- 사전 기간 시각적 추세 없음

**이것이 GO/NO-GO 관문. 다른 어떤 분석보다 먼저 실행.**

---

### 우선순위 2: Oster (2019) Bounds

**구현:**
```
1단계: β_비통제, R²_비통제 (통제변수 없이)
2단계: β_통제, R²_통제 (전체 통제변수)
3단계: R_max = min(1, 1.3 × R²_통제)
4단계: δ* = [β_통제 × (R_max - R²_통제)] / [(β_비통제 - β_통제) × (R²_통제 - R²_비통제)]
```

**기준:** δ* > 1이면 비관측 변수가 모든 관측 변수 합계를 초과해야 함.

**한 문장 무기:** "비관측 선택이 관측 공변량의 [X]배여야 추정 효과를 무효화할 수 있다."

---

### 우선순위 3: 처리 강도 / 용량-반응

**변수:**
```
AIShare_i = (라이더 i의 AI 배정 주문 수) / (사후 기간 전체 주문 수)
```

**핵심 논거:** 단조적 용량-반응은 순수 선택으로 설명하기 매우 어려움 — 비관측 능력이 사용 강도에 정확히 비례해야 한다는 훨씬 강한 가정 필요.

**주의:** 용량 자체가 내생적. "메커니즘 증거"로 프레이밍.

---

### 우선순위 4: 안정 근로자 하위표본

**정의:** 주간 근무일, 일일 근무시간, 서비스 지역, 주문량이 도입 전후 ±1 SD 이내인 라이더

**논거:** "동시 행동 변화" 대안 설명을 직접 무력화.

---

## 근본적으로 해결 불가능한 것 (솔직한 평가)

### 해결 불가한 한계
1. **동시 행동 변화** — 채택자가 AI 채택과 동시에 노력/전략을 변경하면 어떤 매칭/차분도 AI 효과 분리 불가. 도구변수 없음.
2. **짧은 기간** — 어떤 기법도 단기 데이터에서 장기 증거를 생성 불가.
3. **이득에 대한 선택** — ATT ≠ ATE. 채택자가 평균 라이더보다 더 많이 혜택 받을 수 있음.

### 프레이밍 방법
- **해야 할 것**: 명확히 인정, 정량적으로 한정, 왜 발견이 여전히 유익한지 설명
- **해야 할 것**: ATT를 정책 관련 추정량으로 제시 ("플랫폼은 채택할 라이더에 관심")
- **해야 할 것**: 짧은 기간을 한계로 명시 + 장기 패널 복제 연구 촉구
- **하지 말 것**: PSM+DID가 내생성을 "해결한다"고 주장
- **하지 말 것**: 증거 없이 선택을 있을 수 없다고 치부

## Response Letter 템플릿

> 자발적 AI 채택이 제기하는 식별 도전에 대한 리뷰어의 세심한 주의에 감사드립니다. 식별 전략을 대폭 확장했습니다:
>
> **Event-study 분석.** 동적 DID 추정치는 도입 전 계수가 개별적으로, 결합적으로 비유의함을 보여(F-검정 p = [값]), 병행 추세가 유지되었다는 직접 증거를 제공합니다.
>
> **비관측 선택 한정.** Oster (2019) bounds는 δ* = [값]을 산출하여, 비관측 선택이 관측 공변량의 [값]배여야 효과를 무효화할 수 있음을 나타냅니다.
>
> **처리 강도와 용량-반응.** AI 사용 사분위에 걸친 단조적 용량-반응 관계 — 순수 선택으로 설명하기 어려움.
>
> **안정 근로자 하위표본.** 일관된 도입 전후 근무 패턴을 가진 라이더로 제한해도 결과 강건.
>
> 설계가 모든 형태의 시변 비관측 선택을 결정적으로 배제할 수 없음을 인정합니다. 그러나 이 네 가지 보완 분석의 수렴은 그럴듯한 대안 설명의 범위를 상당히 좁힙니다.
